Role: Full Stack Mobile Developer (FinTech & Automation Specialist).

Main Tasks: Developing a secure mobile application (iOS/Android) with dynamic loan application forms and document upload modules; architecting a custom "Bank Matching Engine" that filters eligibility based on salary, debt-to-burden ratios, and employment type; building a multi-bank comparison tool; and creating a robust Admin Dashboard to manage interest rate formulas, insurance fees, and multi-stage request tracking.

Technical Skills: * Mobile: Cross-Platform Development, Secure Document Handling/Storage, Push Notification Integration, and Complex Form State Management.

Backend: Logic-based Matching Engines, Financial Calculation Algorithms (Flat vs. Declining Interest), Automated Eligibility Filtering, and Secure PDF Generation (for Approval Letters).

Security: OTP Authentication, Data Encryption for Financial Documents, and Role-Based Access Control (RBAC).

Tools: Flutter or React Native; Node.js, Python, or Go (Backend); PostgreSQL or MongoDB; Firebase (for OTP & Push); and AWS S3 or Google Cloud Storage (for secure document hosting).

Soft Skills: Strong understanding of FinTech workflows, high attention to mathematical accuracy in interest logic, proactive communication regarding system edge cases, and a commitment to data privacy.
