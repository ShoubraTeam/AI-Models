Role: Web Data Analyst (CRO Specialist).

Main Tasks: Evaluating website performance metrics, identifying conversion bottlenecks, analyzing user behavior, and developing actionable strategies to improve conversion rates.

Technical Skills: Web Analytics, Conversion Rate Optimization (CRO), User Behavior Modeling, Data Interpretation.

Tools: Analytics tools (like Google Analytics, Hotjar, or Mixpanel).

Soft Skills: Ability to turn raw data into strategic insights, "knack for storytelling with data," and identifying areas for improvement.
