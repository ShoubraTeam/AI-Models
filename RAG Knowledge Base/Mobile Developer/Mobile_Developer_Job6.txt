Role: Mobile App Developer (E-commerce & Scalable Systems Specialist).

Main Tasks: Developing a cross-platform (iOS/Android) e-commerce MVP, implementing secure user authentication and profile management, building a dynamic product catalog with advanced filtering (Mobiles, Accessories, etc.), architecting a frictionless checkout and cart management flow, and ensuring the codebase is API-ready for future third-party integrations.

Technical Skills: Cross-Platform Development (Flutter/React Native), Scalable System Architecture, API Integration & Development, State Management (for cart/orders), Secure Authentication (OAuth/Firebase), and E-commerce UX Optimization.

Tools: Flutter or React Native; Firebase or Supabase (for scalable backend/auth); Git; and Figma (for UI implementation).

Soft Skills: Conversion-oriented design thinking, strong architectural problem-solving, clear communication regarding technical milestones, and a commitment to delivery within a structured MVP scope.
