Role: React Native Mobile Developer (MVP & Cross-Platform Specialist).

Main Tasks: Developing a cross-platform MVP using React Native and Expo, prioritizing core functional features over enterprise polish, integrating with simple Node.js backends and APIs, making pragmatic architectural trade-offs, and handling basic App Store and Google Play deployment workflows.

Technical Skills: Cross-Platform Mobile Development, MVP Prototyping, API & Backend Integration (Node.js), Mobile App Security & Authentication, and Basic Native Bridge Knowledge (Swift/Kotlin).

Tools: React Native, React Native Expo, JavaScript/TypeScript, Node.js, Git, and App Store Connect / Google Play Console.

Soft Skills: Pragmatic decision-making, clear communication on technical trade-offs, ability to work independently in a "lean" startup environment, and a focus on scalability for post-MVP phases.
